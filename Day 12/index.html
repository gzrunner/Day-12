Cascading style sheets is a cascading language used for desribing the presentation of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside Javascript and HTML.

Tailwind CSS is an open source CSS framework. The main feature of this library is that, unlike other frameworks like bootstrap, it does not provide a series of predefined classes for elements such as buttons or tables.

Bootstrap is a free and open-source framework directed at responsive, mobile-first front end web development.
